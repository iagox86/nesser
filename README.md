# Nesser

A DNS client and server class, written for Dnscat2 (and other similar projects).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nesser'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nesser
